**1.获取Json Web Token（JWT）**

* 地址：https://app165.acapp.acwing.com.cn/api/token/
* 方法：`POST`
* 是否验证jwt：否
* 输入参数：
  * `username`: 用户名
  * `password`: 密码
* 返回结果：
  * `access`: 访问令牌，有效期5分钟
  * `refresh`: 刷新令牌，有效期14天

---

**9.注册账号**

* 地址：https://app165.acapp.acwing.com.cn/myspace/user/
* 方法：`POST`
* 是否验证jwt：否
* 输入参数：
  * `username`: 用户名
  * `password`：密码
  * `password_confirm`：确认密码
* 返回结果：
  * `result: success`
  * `result: 用户名和密码不能为空`
  * `result: 两个密码不一致`
  * `result: 用户名已存在`
